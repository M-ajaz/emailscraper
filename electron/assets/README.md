Place the following icon files in this folder:

- `icon.png`  — 256x256 PNG, used for Linux window icon, splash screen, and system tray
- `icon.ico`  — Windows multi-size ICO (16, 32, 48, 256px), used for .exe and NSIS installer
- `icon.icns` — macOS ICNS format, used for .app bundle
- `dmg-background.png` — 540x380 PNG (optional), used as macOS DMG installer background
